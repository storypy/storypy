StoryPy
=======

StoryPy is a Python module for metaprogramming.

With StoryPy, you can dynamically create entities (e.g. variables, functions) in your Python program.

For more information about installing and using StoryPy, refer to `the StoryPy documentation <https://storypy-docs.readthedocs.io>`__.
